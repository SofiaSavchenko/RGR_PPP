## Датасет _"Amazon Top 50 Bestselling Books 2009 - 2019"_ содержит 7 столбцов данных:

*   **Name** - название книги
*   **Author** - автор книги
*   **User Rating** - рейтинг книги
*   **Reviewers** - количество рецензий на книгу
*   **Price** - стоимость книги
*   **Year** - год популярности книги
*   **Genre** - жанр книги
